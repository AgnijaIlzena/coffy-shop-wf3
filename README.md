# coffy-shop-wf3
Class project WebForce3.  

Projet Coffee Shop. 
Projet de classe réalisé dans l'objectif de renforcer la compréhension de l'architecture MVC. 
Le site contient une présentation visuelle attirante avec des effets de couleur dynamiques sur les cartes de produits.
Le contenu du site est facilement modifiable par une personne disposant d'un accès administrateur. 
Authentification d'administrateur avec code d'accès pour la session.
